미설정 (기본 다운로드 폴더)

최종 대상 CSV 내보내기

필터링 결과(Tab 4)의 최종 대상을 CSV 파일로 내보냅니다. UTF-8 BOM으로 저장되어 엑셀에서 한글이 정상 표시됩니다.

2차 후보 CSV 내보내기

교육은 종료되었으나 프로젝트가 미마감된 2차 후보를 CSV 파일로 내보냅니다.

보고서 통합본 생성

외부 CSV/XLSX 파일(집체, 원격 등)을 업로드하면 시스템 데이터와 매칭하여 프로젝트코드, 과정명, 에코여부, 강사명, IM명 등을 보완한 통합 보고서를 생성합니다.

전체 데이터 백업

localStorage의 전체 데이터를 JSON 파일로 백업하거나 복원합니다.

데이터 보정

eduDelivery(과정 마스터)와 educationType(운영 상세) 간 불일치를 탐지하고 자동 보정합니다.

데이터 초기화

모든 데이터(과정, 운영현황, 발송이력 등)를 삭제합니다. 복원할 수 없으니 먼저 백업하세요.
