# archive

운영에 사용되지 않는 옛 버전/백업. 참고 목적으로만 보존.

| 경로 | 내용 |
|---|---|
| `version/` | 초기 SPA 단일 HTML 버전 히스토리 (0.01 ~ 0.13) |
| `backup/` | 운영자 수기 백업 JSON/CSV (개인정보 포함되지 않은 집계만) |
| `v2/` | v2 구조 시도 흔적 (단일 index.html) |
| `index_0330.html` ~ `index_v2_backup.html` | 점진 변경 중 만들어진 백업 HTML |
| `cs대상자관리_통합.html` | 초기 통합 버전 (267KB, 단일 페이지) |

운영 코드는 `apps/cs-master/index.html` 과 `apps/cs-master/cs_dashboard.html`. 이 폴더의 파일들은 **history 추적/롤백 참고용**.
